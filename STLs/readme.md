The recomeneded settings for the main body are 
- Automatic tree supports
- 14mm outer brim
- Any infill and wall count (I used 15% gyroid with 3 walls)

I found the optimal printing angle for the main body was a 45* angle, orient the model so that the fully chamfered corner is the only part touching the buildplate then add 2 large manual support dots to the side for support.

<img src="recommended printing orientation for main body.png" width="500"/> <img src="recommended manual support placement.png" width="500"/>

The other pieces are far less picky with their placement so you can just lay them flat on their largest face

The recomeneded settings for everything else is 
- No supports
- No brim
- 20% gyroid infill and 3 walls (feel free to use more of either though)
